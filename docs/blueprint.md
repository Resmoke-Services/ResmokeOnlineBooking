# **App Name**: Resmoke Online Booking

## Core Features:

- Contact Information Form: First page to collect user contact information (name, surname, cell number, email address) and their suburb using a dropdown menu.
- Date & Time Selection: Second page to allow users to select an available date and time for their booking based on data from a POST webhook.
- Booking Confirmation & Notification: Third page displaying a booking confirmation with a notification popup once the booking is finalized.

## Style Guidelines:

- Primary color: A deep, earthy red (#8B0000), reminiscent of fire and the essence of Resmoke's offerings. It conveys warmth and passion.
- Background color: Light beige (#F5F5DC), creating a neutral and inviting canvas that ensures readability and a comfortable user experience.
- Accent color: Muted gold (#B8860B), strategically used to highlight key interactive elements and calls to action, lending an air of sophistication.
- Clean and readable sans-serif fonts.
- Use a clear and intuitive layout with distinct sections for each step of the booking process. Prioritize ease of use and mobile responsiveness.
- Subtle transition animations between pages to enhance user experience
